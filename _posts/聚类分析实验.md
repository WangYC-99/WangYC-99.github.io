## 0. 参数介绍

| 参数     | 解释                         |
| -------- | ---------------------------- |
| nmi      | 归一化互信息评估             |
| ari      | 调整后的兰德指数             |
| si_score | 平均轮廓系数                 |
| ine      | 样本距离最近的聚类中心的总和 |
| tARI     | 调整后的兰德指数             |
| tAMI     | 调整后的互信息               |
| thomo    | 同质化得分                   |
| tcomp    | 完整性得分                   |
| tv_m     | V-measure得分                |
| tsilh    | 平均轮廓系数                 |
| tc&h     | Calinski 和 Harabaz 得分     |
